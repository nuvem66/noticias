1. Instalando o GIT

Windows:

Baixe o executável do GIT e realize a instalação normalmente, sem alterar as opções padrão.

Ubuntu:

No terminal:
$ sudo apt install git Não copie o "$". Outras distribuições Linux.

2. Verificando se o GIT está instalado corretamente

Windows:

Clique com o botão direito na área de trabalho e selecione git bash ou abra o Windows Power Shell. Digite:
$ git --version

Linux:

No terminal: $ git --version

Ambos devem resultar em algo parecido com:


A versão pode variar.

3. Configurando o GIT

Caso não possua, criar conta no GitHub.
Execute os seguintes comandos, um por um: $ git config --global user.name *SEU NOME DE USUÁRIO*
$ git config --global user.email *EMAIL-DO-GITHUB@EXEMPLO.COM* Lembre-se de substituir os campos. O nome de usuário NÂO precisa ser o mesmo do GitHub. No seu computador pessoal, o processo será realizado apenas uma vez. Caso contrário, você pode repetir o processo (ex.: computadores dos laboratórios).

4. Clonando repositórios do GitHub usando o GIT

Crie um novo repositório no GitHub chamado "noticias": Escolha um diretório para clonar os aquivos. Recomenda-se criar uma pasta chamada workspace em C:/ (Windows) ou /home/*NOMEDOUSUARIOLINUX*/ (Linux). Abra essa pasta no terminal (use o comando cd para navegar até o diretório ou abra um gerenciador de arquivos e, ao clicar com o botão direito, "Abrir terminal/cmd").
Copie o link do seu repositório: Com o link do seu repo em mãos, você pode cloná-lo com o seguinte comando: $ git clone *linkdorepositorio* Exemplo (meu repositório): $ git clone https://github.com/nuvem66/noticias.git Tenha certeza de que você está na pasta correta, pois a clonagem ocorre lá!

6. Sincronizando VSCode e GitHub usando o GIT

Abra seu na pasta onde você realizou a clonagem do repositório. Clique no ícone de usuário na sua barra lateral, Backup and Sync Settings Clique em Sign in. Clique em Sign in with GitHub. Uma página será aberta no seu navegador. Realize os processos necessários. Abra o link em seu VSCode. Pronto! Ao clicar no ícone de perfil novamente, você pode ver sua conta. Você pode enviar seus arquivos ao clicar em Commit e sincroniza-los com o repositório clicando em Sync Changes.
